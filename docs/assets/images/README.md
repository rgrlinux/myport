# Diretório de Imagens

Coloque suas imagens aqui:
- hero-background.jpg
- logo.png
- profile.jpg
- etc.

Formato recomendado:
- WebP para performance
- JPEG para fotos
- PNG para logos com transparência
- SVG para ícones e ilustrações

Tamanhos recomendados:
- Hero images: 1920x1080px (WebP ~150KB)
- Thumbnails: 400x400px (WebP ~30KB)
- Icons: 64x64px (SVG)
